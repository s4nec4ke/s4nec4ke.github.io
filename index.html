Magic Pot

Decentralised prize-pool game on TON — Telegram Mini App + smart contract

t.me/themagicpot ↗

01What & why

In 2024 the TON ecosystem was growing fast — but project communities were weak or bot-inflated, dragging token prices down. Magic Pot lets any token creator launch a customisable on-chain prize pool for token holders in five minutes: flexible winner mechanics, multi-token pools — all via smart contract, directly inside Telegram chats.

В 2024 году экосистема TON активно росла — но комьюнити проектов было слабым или состояло из ботов, оказывая негативное влияние на цену токенов. Magic Pot позволяет любому создателю токена запустить для токенхолдеров настраиваемый on-chain пул с игрой за пять минут: гибкие механики победителей, мультитокенные пулы — всё через смарт-контракт, прямо в чатах Telegram.

02What I built

  • Telegram Mini App with custom game mechanics
  • Smart contract on TON for draws and prize pools
  • 2-click onboarding — join a game from chat with no sign-up
  • Visual identity from scratch
  • Bot with live in-chat commentary for draws
  • Telegram Mini App с уникальными механиками
  • Смарт-контракт на TON для розыгрышей и призовых пулов
  • Онбординг за 2 клика — участие в игре из чата без регистрации
  • Визуальная идентичность продукта с нуля
  • Бот с live-комментированием розыгрыша в чате

03Team & how I led it

  • 6-person team: backend, smart-contract dev, DevOps, frontend, illustrator, community manager
  • Scrum + Kanban in Notion
  • Hired through the early TON ecosystem — closed 2 replacements within a week without a sprint slip
  • Total budget: $20K over the project lifespan
  • Команда 6 человек: бэкенд, разработчик смарт-контрактов, DevOps, фронтенд, иллюстратор, модератор сообщества
  • Scrum + Kanban в Notion
  • Нанимал через раннюю экосистему TON — закрыл 2 замены за неделю без сдвига спринта
  • Общий бюджет: $20K за время жизни проекта

04Distribution & traction

  • 20+ partnerships with crypto community leaders
  • Each new partnership: +50% MAU
  • 40% of users in active partner chats participated in games
  • 20+ партнёрств с лидерами крипто-комьюнити
  • Каждое новое партнёрство: +50% MAU
  • 40% пользователей в активных партнёрских чатах участвовали в играх

05TON Accelerator pitch

Built the deck in English solo. Pitched on Twitter Spaces in front of a large audience and the accelerator team. Outcome: $5,000 grant from the TON Foundation.

Собрал презентацию на английском самостоятельно. Выступил в Twitter Spaces перед большой аудиторией и командой акселератора. Результат: грант $5,000 от TON Foundation.

case · 03 · ai_jobsearch
⚡ building · 2026 · side project

AI Job Search Agent

A personal Claude Code agent for the rest of this job hunt

github: s4nec4ke/hr-agent-ru ↗

01The brief I gave myself

Spending 2 hours a day reading Telegram job channels is painfully boring and kills motivation. Manually tracking outreach is even worse. I built an agent that handles the routine — parse, score, draft outreach — and gives me back the time for interview prep.

Тратить 2 часа в день на чтение Telegram-каналов с вакансиями — очень скучно и негативно влияет на мотивацию. Систематизировать вручную работу с откликами — ещё скучнее. Я собрал агента, который делает рутинную часть — парсит, оценивает, пишет черновик аутрича — и возвращает мне время для подготовки к интервью.

02Architecture

  • hr-collect — Telethon parses configured Telegram channels, runs keyword pre-filtering, dedupes, drops vacancies into a per-day folder
  • cv-match — LLM scores each vacancy against my candidate profile and picks the resume with the best-matching keywords for the application
  • vacancy-card — generates a one-pager per vacancy: card + draft HR message
  • interview-prep — when a vacancy hits "interview scheduled", generates brief + script + transcript template
  • hr-daily — orchestrator agent, runs the whole chain end-to-end
  • Thin CSV index (10 columns) + per-vacancy folders for long text — fast eyeball scanning, no DB overkill
  • hr-collect — Telethon парсит настроенные Telegram-каналы, проводит первичную фильтрацию по ключевым словам, дедуплицирует, складывает вакансии в папку за день
  • cv-match — LLM оценивает каждую вакансию по профилю кандидата и подбирает для отклика резюме с наиболее подходящими ключевыми словами
  • vacancy-card — генерирует одностраничную карточку на вакансию: карточка + черновик сообщения HR
  • interview-prep — когда вакансия получает статус "интервью назначено", создаёт brief + script + шаблон транскрипта
  • hr-daily — агент-оркестратор, запускает всю цепочку от начала до конца
  • Тонкий CSV-индекс (10 колонок) + папки на каждую вакансию для длинных текстов — быстрое визуальное сканирование без лишней сложности

03Stack

  • Python + Telethon for the Telegram side
  • Claude Code skills as the orchestrator
  • Local CSV + filesystem (no DB — overkill at this size)
  • Python + Telethon для Telegram-части
  • Claude Code скиллы как оркестратор
  • Локальный CSV + файловая система (без БД — избыточно на таком объёме)
